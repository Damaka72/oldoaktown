# Old Oak Town

**Your comprehensive guide to West London's largest regeneration project**

[![RSS Aggregation](https://github.com/Damaka72/oldoaktown/actions/workflows/rss-aggregation.yml/badge.svg)](https://github.com/Damaka72/oldoaktown/actions/workflows/rss-aggregation.yml)

## 📍 About

Old Oak Town is a hyperlocal news and community resource dedicated to tracking the Old Oak Common regeneration project in West London. Covering the HS2 station development, new housing, jobs, and community developments, we serve as the authoritative source for residents, businesses, and stakeholders.

## 🎯 Project Overview

- **Scale:** 140 hectares regeneration zone
- **Impact:** 25,500 new homes, 65,000 new jobs
- **Timeline:** 2025-2039+ (long-term project)
- **Focus:** Hyperlocal journalism, community engagement, business directory

## 🏗️ Strategic Approach

This project follows a comprehensive strategic blueprint designed to build a high-value digital asset. Key elements:

1. **Hybrid Business Model:** Aggregator → Marketplace evolution
2. **Three-Pillar Content Strategy:**
   - Factual & Foundational (70% - automated with human oversight)
   - Hyperlocal Lifestyle (20% - community engagement)
   - Deep Dive Analysis (10% - investigative journalism)
3. **Diversified Revenue:** Ads, directory listings, affiliates, memberships
4. **Human-in-the-Loop:** Automation with quality control
5. **Long-term Asset Building:** Patient strategy for 2039 timeline

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone repository
git clone https://github.com/Damaka72/oldoaktown.git
cd oldoaktown

# Install dependencies
npm install

# Run RSS aggregation
npm run fetch-feeds
```

### Daily Workflow

1. **Automated RSS Aggregation** (GitHub Actions runs daily at 8 AM UTC)
2. **Review Queue** - Check `admin/review.html` for pending content
3. **Human Review** - Fact-check, edit, approve/reject articles
4. **Publishing** - Approved content goes live
5. **Analytics** - Monitor traffic and engagement

## 🛠️ Tech Stack

- **Frontend:** Static HTML/CSS/JavaScript (fast, secure, cheap hosting)
- **Content:** RSS aggregation + human curation
- **Automation:** GitHub Actions for daily feeds
- **Analytics:** Google Analytics 4
- **Monetization:** Google AdSense, direct advertising, affiliates
- **Hosting:** Netlify/Vercel (recommended) or GitHub Pages

1. **Programmatic Ads** (Google AdSense) - £100-300/month
